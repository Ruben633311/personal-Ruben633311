voor deze opdracht heb ik een systeem bedacht waar ik een system context diagram voor ga maken.
Ik kies voor een minecraft server systeem. 

Key drivers:

| stakeholder     | beschrijving | Key drivers        |
|:---------|:-----:|-------------------:|
| users    | de users zijn mensen die de minecraft server gebruiken om te spelen. ze proberen verder te komen in het spel.    | blijdschap, verbondenheid   |
| host     | de host is het persoon dat de server heeft opgericht, hij heeft de server ook zo ingesteld dat meerder mensen kunnen spelen.     | blijdschap, macht, exhibitionisme |
| admin  | de admin is het persoon dat orde in de server wilt behouden     | veiligheid, orde |
| mojang  | mojang is het bedrijf dat minecraft maakt  | macht, geld, data |

| F1    | Users kunnen minecraft op de server spelen. |
|:---------|:-----:|
| rationale | Users willen minecraft spelen omdat het leuk is |
| MoSCoW | must have |

| F2    | Users kunnen samen op dezelfde server spelen |
|:---------|:-----:|
| rationale   | de users willen contact hebben met anderen in het spel |
| MoSCoW | must have |

| F3    | De host moet controle hebben over de server |
|:---------|:-----:|
| rationale    | de host maakt de minecraft server, omdat het leuk is om samen minecraft te spelen |
| MoSCoW | must have |

| NF1    | de admin moet macht hebben in de server door commands te kunnen gebruiken |
|:---------|:-----:|
| rationale    | de admin wil orde hebben in de server door commands te gebruiken  |
| MoSCoW | should have |

| NF2    | mojang maakt geld en krijgt macht |
|:---------|:-----:|
| rationale    | mojang wil geld en macht door mensen die minecraft kopen om op een server te spelen.  |
