Voor deze STD code opdracht heb ik switch case code gemaakt en uiteindelijk flags toegevoegd in de code om de knoppen beter te detecteren als ze ingedrukt worden.

Bij deze STD staan de switch cases en wordt gecheckt of een knop wordt ingedrukt en als dat gebeurd dan gaat hij naar de volgende state zoals het is geprogrammeerd.
![alt text](../foto's/STD_trafficlight.png)

In plaats van alleen te checken of een knop is ingedrukt wordt een flag gezet wanneer een knop wordt gedrukt en dan wordt er bij elke loop steeds gecheckt of een flag is gezet en als dat waar is dan gaat het programma naar de volgende state. Zo hoef je de knop niet ingedrukt te houden totdat het programma het detecteerd zoals bij de code dat ik hiervoor heb gemaakt
